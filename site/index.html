Microsoft Graph source collection for agents

Graphclaw collects Microsoft 365 sources for your agents.

Outlook mail, calendars, and contacts become durable local SQLite mirrors with raw provenance and deterministic JSONL/Markdown exports for agent memory.

View on GitHub Install locally 
graphclaw org add company --tenant organizations --client-id ...
graphclaw auth login --org company
graphclaw sync mail --org company --folder inbox --json
graphclaw export markdown --org company --out ./agent-sources/microsoft --json

Not an admin CLI. A memory ingestion layer.

Graphclaw is for long-running local mirrors where cursors, provenance, deletes, and company boundaries matter more than one-off API calls.

Local-first

Each company gets its own SQLite mirror.

Raw Graph payloads, sync runs, projections, and OAuth state live under isolated org folders.

Delta-native

Resume from Graph delta links.

Mail, calendar, and contacts sync from saved cursors instead of repeatedly listing everything.

Agent-ready

Export clean evidence, not loose summaries.

JSONL and Markdown exports preserve source links so agents work from deterministic data.

From Microsoft Graph to agent-ready sources in five deterministic steps.

Code handles the mechanical parts. The agent gets structured evidence and does judgment work after collection.

1Graph API

Read-only delegated scopes for Microsoft 365 data.

2Delta cursor

Store the exact resume token for each sync scope.

3SQLite mirror

Keep raw JSON and normalized projections per org.

4Exports

Generate JSONL and Markdown source packs.

5Enrichment

Your agents turn evidence into searchable working memory.

Install, connect, sync.

Graphclaw uses Microsoft device-code auth, so setup works cleanly from a terminal without embedding browser automation.

git clone https://github.com/andrewnova/graphclaw.git
cd graphclaw
python3 -m pip install -e .
graphclaw doctor

graphclaw org add personal --tenant common --client-id YOUR_ENTRA_APP_CLIENT_ID
graphclaw auth login --org personal
graphclaw list mail-folders --org personal --json
graphclaw sync calendar --org personal --start 2026-05-01T00:00:00Z --end 2026-06-01T00:00:00Z --json
microsoft-graph outlook calendar contacts sqlite agent-sources local-first
Read-only by default: User.Read, Mail.Read, Calendars.Read, Contacts.Read, offline_access.
